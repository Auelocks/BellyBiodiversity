Belly Button Biodiversity Dashboard

This page is designed to visualize the bacterial data for different bacterial species found in the belly buttons of human volunteer test subjects. Each individual test subject is identified by Test Subject ID Number. The Demographic Info for each test subject, Top 10 Bacteria Cultures Found in subjects’ belly buttons (including names and counts), and the test subjects’ reported washing frequency can be isolated and viewed by selecting Test Subject ID No. from the dropdown menu below.
